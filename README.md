# Group Elements of Same Indices

A Python script that groups elements from multiple lists by their index position (matrix transposition).

## Example

```python
[[10, 20, 30], [40, 50, 60], [70, 80, 90]]
# → [10, 40, 70]  [20, 50, 80]  [30, 60, 90]
```

## Technologies

- Python 3

## Getting Started

```bash
python code.py
```
